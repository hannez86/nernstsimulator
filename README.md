# Nernst-Gleichung Simulator

Interaktive HTML5-Simulation zu elektrochemischen Zellen und der vereinfachten Nernst-Gleichung.

## Features

- Auswahl verschiedener Elektroden-Metalle und Elektrolyte
- Variable Konzentrationen (0.0001–2.00 M)
- Visualisierung der Zellspannung mit Voltmeter
- Aufgaben/Level inkl. Konzentrationsaufgaben (M1-Auswertung)
- Modals für Hintergrund, Standardpotentiale und Lizenz

## Nutzung

1. `index.html` im Browser öffnen.
2. Elektroden und Lösungen wählen.
3. Konzentrationen einstellen und messen.

## Projektstruktur

- `index.html` – UI und Styles
- `simulation.js` – Logik der Simulation
- `simulation.html` – ältere Referenzversion
- Weitere `*.htm` – historische Inhalte/Quellen
